#-------------------------- GENERAL SETUP ------------------------------
An R script for plotting 2D projections of samples and estimate the 2D and marginal density is given in the file plot_samples.R. To make it work:
	o	set user-defined variables (like labels for the plot and source file)
	o	run with `Rscript plot_samples.R`
	o	the picture will appear in the same folder as the source file and will have the same name with the corresponding extension

#-------------------------- LIBRARIES ----------------------------------
If the "sp" and "MASS" libraries are not installed, run
install.packages("sp")
install.packages("MASS")

#---------------------------- EXAMPLE -----------------------------------
o	Run `Rscript plot_samples.R` from the current directory
o	The output file curgen_db_007.png should be produced
